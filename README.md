# sketch-to-image-gan
Generates high-resolution, realistic images from sketches using GAN-based architectures trained on paired sketch-image datasets.

# Image Generation from Sketches using GANs

## 📌 Problem
Generate high-resolution realistic images from grayscale or binary sketches.

## ⚙️ Tech Stack
- Python
- PyTorch
- GANs(cyclic GAN, Pix-2-Pix GAN, Combined architecture+ L1 loss), U-Net

## 🚀 Approach
- Implemented multiple architectures including U-Net and modified GANs
- Trained on sketch-image datasets (CUHK, FERET)
- Compared model performance

## 📊 Results
- Generated high-quality images from sketches
- Identified best-performing architecture

## 💡 Impact
Applications in design, digital art, and forensic reconstruction.
